## 설치 방법

프로젝트를 개발하기 위해 다음 단계를 따라주세요 ☺︎

### 1. 이 프로젝트를 클론 합니다.

```sh
git clone https://github.com/Jy0042/dururu.git
```

### 2. 프로젝트 디렉토리로 이동합니다.

```sh
cd dururu
```

### 3. VSCode를 실행합니다.

```sh
code .
```

### 4. npm 설치 후 실행합니다.

```sh
npm install
```

```sh
npm install
```
